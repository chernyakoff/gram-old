Текущий статус диалога: {current_status}

После ответа выполни OOC-анализ (вне контекста):

  1. Проверь, достигнута ли цель текущего статуса {current_status} на основе всего диалога.
  2. Если цель достигнута — верни новый статус (следующий по порядку: init -> engage -> offer -> closing).
  3. Если нет — верни текущий статус.
  4. Добавь в конец ответа строку: OOC_STATUS: [статус]

Пример: Если в init ты поприветствовал и задал вопрос, а пользователь ответил — цель достигнута, верни engage.
