# Gram-old Timeline Notes

## Контекст

- Это независимый клон текущего проекта.
- Репозиторий привязан к отдельному `origin` на GitHub.
- Цель: использовать репозиторий как "машину времени" для поэтапного прохождения истории изменений.

## Базовая точка отката

- Состояние проекта за конец декабря 2025 зафиксировано на коммите:
  - `15ad0129cd50223cd745dcad26cd8df8c2216617`
  - дата коммита: `2025-12-30 18:37:37 +0300`

## Рабочий процесс

- Основная рабочая ветка для отката и прогресса вперед: отдельная timeline-ветка.
- Дальше движение вперед идет от точки отката по истории коммитов.
- Продвижение выполняется не обязательно по одному коммиту: можно идти пакетами по несколько коммитов.
- После каждого шага/пакета изменений планируется выкладка и тестирование.
